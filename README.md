# NormWM_4_Glioblastoma
Working code to investigate the white matter pathways involved in human glioblastoma to understand patient outcomes.

If you use the code, please cite the following: [https://academic.oup.com/neuro-oncology/advance-article/doi/10.1093/neuonc/noaf234/8287534](https://academic.oup.com/neuro-oncology/advance-article/doi/10.1093/neuonc/noaf234/8287534).

This code is being updated regularly, and it is not yet intended as a standalone solution.
